## About

This repository provides unofficial AppImage builds of the stable Discord Linux client.

The AppImage is built from Discord’s official Linux binary release. No PTB or Canary builds are provided here. This repository is only for the stable Discord release.

## Recommended AppImage manager

For the easiest experience, use [Gear Lever](https://github.com/mijorus/gearlever) to install, manage, and launch this AppImage.

Gear Lever can integrate AppImages into your Linux desktop, so Discord appears in your app launcher like a normal installed application. It also makes managing AppImages cleaner than leaving random executable files in your Downloads folder, which is how chaos starts.

After downloading the AppImage, open it with Gear Lever and let it handle the desktop integration.

## Alternative: zap

You can also install it using [`zap`](https://github.com/Gur0v/zap), the command-line AppImage package manager:

```bash
zap install --github --from=Gur0v/discord-appimage discord-appimage
````
